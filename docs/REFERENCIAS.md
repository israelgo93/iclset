# REFERENCIAS.md

## Propósito

Este archivo reúne referencias de sitios modernos de conferencias, plataformas académicas y repositorios científicos útiles para diseñar y desarrollar el sitio oficial de ICLSET 2026.

## Plataformas científicas y repositorios recomendados

### LA Referencia

URL: https://www.lareferencia.info/es/

Uso:

- Red latinoamericana de repositorios de acceso abierto.
- Útil para buscar producción científica regional.
- Relevante para referencias en ciencias biológicas, ambientales, agrociencias y tecnologías.

### Latindex

URL: https://www.latindex.org/

Uso:

- Sistema regional de información sobre revistas científicas de América Latina, el Caribe, España y Portugal.
- Útil para verificar revistas, indexación y referencias editoriales regionales.

### SciELO Ecuador

URL: http://scielo.senescyt.gob.ec/

Uso:

- Biblioteca científica electrónica de Ecuador.
- Referencia para acceso abierto y producción científica ecuatoriana.

### RRAAE

URL: https://rraae.cedia.edu.ec/

Uso:

- Red de Repositorios de Acceso Abierto del Ecuador.
- Útil para localizar tesis, artículos y documentos institucionales ecuatorianos.

### Dialnet

URL: https://dialnet.unirioja.es/

Uso:

- Portal bibliográfico con revistas, artículos, tesis y literatura científica iberoamericana.
- Útil para validación editorial y visibilidad académica.

### DOAJ

URL: https://doaj.org/

Uso:

- Directorio de revistas de acceso abierto.
- Útil para identificar revistas revisadas por pares y buenas prácticas editoriales.

### OpenAlex

URL: https://openalex.org/

Uso:

- Base abierta de publicaciones, autores, instituciones, conceptos y revistas.
- Útil para investigación bibliométrica y análisis de temas por track.

### IEEE Xplore

URL: https://ieeexplore.ieee.org/

Uso:

- Biblioteca científica y técnica para ingeniería, informática, software, IoT y ciberseguridad.
- Relevante para Track 3.

### ACM Digital Library

URL: https://dl.acm.org/

Uso:

- Fuente clave para software, HCI, sistemas, IA y computación.
- Relevante para Track 3.

### AGRIS FAO

URL: https://agris.fao.org/

Uso:

- Base internacional de agricultura y ciencias relacionadas.
- Relevante para Agrociencias y sostenibilidad.

## Aplicación en la versión inicial · 2026-05-11

- De ICLR, NeurIPS y CVPR se adapta la separación entre home, Call for Papers, programa, comités y fechas importantes.
- De CLEI, CITIS e INCISCOS se adapta el enfoque regional, multitrack y de convocatoria académica.
- De las referencias técnicas se implementa App Router, SSG por locale, metadata, sitemap, robots, Tailwind CSS v4, shadcn/ui y preparación Supabase.
- Las URLs externas específicas de plantillas quedan pendientes hasta confirmación institucional.
- El enlace final de envio CMT queda pendiente hasta que Microsoft cree el sitio oficial, pero el sitio ya publica el placeholder aceptado por CMT y el acknowledgement verificable.
